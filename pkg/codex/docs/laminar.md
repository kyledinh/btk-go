# Laminar for ScalaJS

## Usage

build.sbt
```
"com.raquo" %%% "laminar" % "0.13.1"  // Requires Scala.js >= 1.5.0
"com.raquo" %%% "airstream" % "<version>"
```

import
```
import com.raquo.api.L.{*, given}  // Scala 3
```
